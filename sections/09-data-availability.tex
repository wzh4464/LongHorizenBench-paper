The replication package is available at \url{TBD}.  It includes:
(1)~task descriptions for all 62 benchmark tasks (12 industrial plus 50 CapBench), with per-task metadata recording repository, parent commit, ground-truth commit, handwritten-file list, and natural-language prompt;
(2)~the full set of 124 agent prompts (62 tasks $\times$ 2 prompt variants);
(3)~the rubric and evaluator prompt template used by all four LLM evaluators;
(4)~per-run artifacts for all 496 experiments---final diffs, timing metadata, session logs, and sanitization verification---with per-evaluator scores, composite scores, and verdicts;
(5)~the scoring scripts, aggregation notebooks, and sanitization harness.
We also distribute the per-evaluator scoring CSV (\texttt{eval\_scores\_4evaluators.csv}) used to produce every results table in this paper.
The replication package contains no proprietary source code; only the public upstream repositories referenced by each task are required.
